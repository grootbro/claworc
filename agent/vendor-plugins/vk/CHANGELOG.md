# @openclaw/vk

## 2026.3.30-beta.0

- Initial VK channel plugin scaffold for OpenClaw.
- Added webhook-first configuration surface for VK Community Messages.
- Added account resolution, status summary, and package metadata foundation.
- Added Callback API webhook handler for `confirmation` and `message_new`.
- Added inbound normalization and basic auto-reply dispatch path.
- Added outbound `messages.send` and optional `messages.markAsRead` helpers.
